# 02 · 比赛规则与提交规范

## 一、时间规则

- 每天只能提交**前一个交易日**的结果（数据 T+1 滞后，官方每晚 18 点左右更新昨日答案）。
- 交易日当天 **23:59 前**提交，过期该日计 0 分。
- **A 榜每天最多 3 次提交，取最后一次**（不会增加次数，因股票池小）。
- 周末休市也可提交，评测结果在**第一个交易日 T+1** 更新。
- ⚠️ `transaction_date` 必须填**昨日（前一交易日）**日期，格式错/日期错会报错或 0 分。
- B 榜每个交易日必须提交（据 main.py 头注释 ≥8 次；以官方页面为准）。

## 二、文件格式

打包为 `submit.zip`，内含两个 CSV（**UTF-8 编码，LF 换行**），**不得嵌套文件夹**。
两文件字段的**顺序、名称必须完全一致**，不得修改/新增/删除字段。

### pattern_reco.csv — 交易模式识别
```
stock_code,transaction_date,pattern_type,pattern_explanation
603316,20260630,大单吸筹,资金大笔挂单买入短时间内集中扫货
```
- `stock_code`：纯数字，**去掉 `.SH`/`.SZ` 后缀**
- `transaction_date`：前一交易日，`YYYYMMDD`
- `pattern_type`：模式类别名称（可自由打标，不限个数）
- `pattern_explanation`：模式说明文字
- **100 行**（对应 100 只股票；停牌股票可删除该行或随意填充）

### predict_result.csv — 资金类型与意图识别
```
stock_code,transaction_date,capital_type,capital_intention
603316,20260630,游资,买入
```
- `capital_type`：`游资` / `量化` / `散户` **三选一**（其他值报错）
- `capital_intention`：`买入` / `卖出` / `T0交易` **三选一**
- **100 行**

> ⚠️ 编码提醒：CLAUDE.md 明确要求 **UTF-8 + LF**。而 `main.py`/baseline 教程用的是
> `utf-8-sig`（带 BOM）。以官方赛题页要求为准，**建议用不带 BOM 的 UTF-8**，避免 BOM 干扰解析。
> 换行统一 LF（Windows 下注意 `\r\n` → `\n`）。

## 三、评分规则

**A/B 榜总得分 = 交易模式识别分 × 0.4 + 参与者识别分 × 0.6**

### Task1 交易模式识别（40%）
基于四大指标评估聚类的类内聚合度与类间区分度，**针对选手自己聚出的 `pattern_type` 标签计算**
（同标签样本算聚合度，不同标签样本算区分度）：
1. **轮廓系数**（Silhouette）：[-1,1]，越接近 1 越好。
2. **CH 指数**（Calinski-Harabasz）：类间/类内方差比，越高越好。
3. **Wasserstein 距离**：不同模式的分布差异，越大区分度越好。
4. **DTW 时序距离**：不同交易行为的时间序列差异，越大区分度越好。

> A 榜**不评** `pattern_explanation`；B 榜将其纳入可解释性综合评分。

### Task2 参与者识别（60%）
- 基于**平台真实资金标签**计算**加权 F1-Score**，**只对 `capital_type`** 打分。
- `capital_intention` **不做标签打分**，仅作 B 榜可解释性评分。
- 辅助指标：精确率（查准）、召回率（查全）。

> 真值来源：需未来若干交易日的实盘回溯才能确定（文档间说法不一：baseline 正文称 T+5、
> main.py 头注释称 T+8）。日常看到的即时分数是「昨日答案更新后」的临时反馈，非最终成绩。

## 四、合规红线

1. **严禁未来函数**：所有特征/模型/规则仅可用当日盘中可获取的 L2 数据，
   不得用当日收盘后或未来交易日的任何数据。
   - 允许：用历史交易日数据训练模型（训练不限）。
   - 禁止：预测所用数据包含未来信息（如含未来的全局归一化）。
2. **严禁硬编码个股标签**：不得针对特定个股/特定日期设固定标签规则，需具备普适性。
3. **严禁使用平台评测真值**。
4. **代码可复现**：B 榜前 15 名需提交完整项目代码，逻辑须与方案报告一致；
   固定随机种子、相对路径、`main.py` 为入口。
5. **禁止小号刷实验**：后台有操作留痕，成绩公示前会做数据清洗与身份复核，关联账号一并取消资格。
