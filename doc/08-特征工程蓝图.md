# 08 · 特征工程蓝图（分类特征字典）

> **唯一目标**：对每只股票单日的**主导资金类型（游资/量化/散户）**做高准确度分类（Task2, 60%），
> 并为交易模式聚类（Task1, 40%）提供有区分力的行为刻画。
> 本文是特征全景地图，后续特征工程围绕它推进——**每加一类特征都必须用区分度/F1 量化验证其价值，无效即砍**。

## 设计原则（避免"复杂但没用"）

1. **对准区分度**：特征存在的理由是能把三类分开，尤其是**量化**（最难：游资有大单单边、散户有小单零散，量化的指纹藏在微观模式里）。
2. **时序 > 聚合**：不要把一天拍扁成一个均值。游资"先吸筹后拉升"、量化"全天均匀高频"是**时间形态**，聚合会抹平。
3. **善用全息数据**：`order.ordr_no` ↔ `transaction.ask_ordr/bid_ordr` 可关联，能重建订单簿、追踪每笔委托生命周期——这是识别量化的金矿。
4. **量化验证**：每类特征加入后，看它是否提升簇分离度（Task1）或 A 榜 F1（Task2）；参考 [07 勘验](07-真实数据勘验结论.md) 的判别检验方法（如"撤单率无区分力"就是这样发现的）。

| 表 | 关键字段 | 粒度 |
|----|---------|------|

## 判别先验（贯穿全文的三类画像）

| 维度 | 游资 | 量化 | 散户 |
|------|------|------|------|
| 单笔规模 | 大单为主、集中 | 小单为主、均匀 | 中小单、随机 |
| 方向性 | 强单边（追涨） | 均衡（做市/套利） | 弱偏向 |
| 频率 | 中低频、择时爆发 | 高频、全天持续 | 低频、零散 |
| 撤单 | 少，以成交为目的 | **多且快**，试探/幌骗 | 少、无策略 |
| 订单规律 | 人为、不规则 | **算法化、高度规律** | 随机、不规则 |
| 流动性 | 消耗（taker） | 提供（maker）为主 | 消耗为主 |
| 时段 | 集中出击 | 全天均匀 | 随机 |

## A. 成交层特征（transaction）

| 特征 | 说明/公式 | 判别指向 | 优先级 |
|------|-----------|---------|--------|
| oss_{mega,large,mid,small}_amount_pct | 按成交额分级（≥50万/10万/4万）金额占比 | 游资↑(mega/large)，散户↑(small) | P0 |
| oss_*_count_pct | 各级笔数占比 | 同上 | P0 |
| trd_avg_trade_amount | 总成交额/笔数 | 游资↑ | P0 |
| trd_total_count | 总成交笔数 | 量化↑ | P0 |
| ap_active_buy/sell_pct | 主买/主卖金额占比（trbsflag） | — | P0 |
| ap_active_net_pct | 净主动占比 (买-卖)/总 | 游资↑(拉升) | P0 |
| ap_unilateral_intensity | \|净主动\|，单边强度 | 游资↑ | P0 |
| ap_active_buy/sell_run_max | 主买/主卖最长连续笔数 | 游资↑ | P1 |
| **成交量 size 分布熵** | -Σp·log p（size 分箱）→ 越低越规律 | **量化↓(规律)**、散户↑ | **P0(攻量化)** |
| **成交量众数集中度** | 出现最多的 size 占比（算法固定手数） | **量化↑** | **P0(攻量化)** |
| **成交量自相关** | 相邻成交 size 的 autocorr（拆单→正相关） | **量化↑** | **P1(攻量化)** |
| **相同 size 重复率** | 连续相同 size 占比 | **量化↑** | P1 |
| rs_interval_cv/skew/kurt | 成交间隔变异/偏度/峰度 | 散户↑(cv) | P1 |
| rs_burst_ratio / hft_ratio | 间隔<1s / <0.5s 占比 | 量化↑ | P0 |
| rs_interval_autocorr | 成交间隔自相关（节奏规律） | 量化↑ | P1 |
| **价格冲击 Kyle's λ** | \|Δ价\|/成交量 回归斜率 | 游资↑(冲击大) | P1 |
| **吃档深度** | 单笔成交吃穿的盘口档数（需对齐 snap） | 游资↑ | P2 |
| **成交后价格恢复** | 大单成交后 N 秒价格回复比例 | 游资↓(持续)、量化↑(临时) | P2 |
| fd_herfindahl | Σ(成交额占比)²，集中度 | 游资↑ | P0 |
| fd_top5_amount_pct | 最大 5% 笔的金额占比 | 游资↑ | P1 |
| fd_amount_entropy | 成交额分布熵 | 散户↑ | P1 |

## B. 委托层特征（order）

| 特征 | 说明/公式 | 判别指向 | 优先级 |
|------|-----------|---------|--------|
