# 01 · 赛题背景与任务

## 一、背景与痛点

构建一套「市场参与者交易行为识别与资金流向分析」体系：结合逐笔成交、订单簿微观结构、
个股基本面等信息，从海量高频数据中自动解析不同参与者的买卖行为与资金流向，识别背后真实意图。

例如：识别订单的资金属性（游资 / 量化私募 / 散户），分析资金意图（吸筹、试盘、对倒、拉升、
出货等），生成清晰的资金流向结论（当前谁占主导、买还是卖、真实意图是什么）。

普通投资者面对 Level-2 数据的两大痛点：
- **信息碎片化**：单指标难以还原机构真实建仓/拉抬/出货意图，机构常用拆单、对倒隐藏动作。
- **解读滞后性**：多数资金流向指标基于日线统计，盘中动态无法捕捉，信号明确时股价已大幅偏离。

> 通俗版：看到某股突然放量大涨——是游资拉涨停准备出货，还是量化在做 T+0 套利？
> 前者该跟，后者该躲。本赛题就是训练 AI 自动回答这个问题。

## 二、两大核心任务

### Task1：交易模式识别（无监督聚类）
- **目标**：基于全量 L2 特征，对单日个股交易行为做无监督聚类，划分有显著区分度的交易模式，
  输出可解释的模式类型与业务说明。
- **要求**：聚类满足「类内高聚合、类间高区分」；模式解释贴合 A 股真实交易逻辑，无业务常识错误。
- **输出**：`pattern_reco.csv`，4 字段固定顺序：
  `stock_code, transaction_date, pattern_type, pattern_explanation`
- **标签自由**：`pattern_type` 可自由打标、不限个数，按合理性与可解释性打分。

### Task2：资金类型与交易意图识别（无真值规则判别）
- **目标**：识别单日个股的主导资金类型，并判断核心交易意图。
- **要求**：
  - `capital_type` **仅**可为「游资 / 量化 / 散户」三类（否则报错）。
  - `capital_intention` **仅**可为「买入 / 卖出 / T0交易」三类。
- **输出**：`predict_result.csv`，4 字段固定顺序：
  `stock_code, transaction_date, capital_type, capital_intention`

> ⚠️ 评分要点（来自官方 Q&A）：A 榜 Task2 **只对 `capital_type` 算加权 F1**；
> `capital_intention` 仅作 B 榜可解释性打分。详见文档 02 / 03。

### 原始数据组织（真实 A/B 榜）
按「日期目录 / 股票代码 / 三类 CSV」层级组织，**GBK 编码、中文列名**：

```
data_root/
├── 20260618/              ← 日期目录
│   ├── 600000/            ← 股票代码目录
│   │   ├── 行情.csv        ← 3 秒 L2 快照（十档盘口量价）
│   │   ├── 逐笔成交.csv     ← 逐笔成交明细（含 BS 标志）
│   │   └── 逐笔委托.csv     ← 逐笔委托明细（含撤单事件）
│   └── ...
└── 股票样本.xlsx           ← 100 只目标股票列表
```

三源文件核心字段：

| 文件 | 关键字段 | 业务含义 |
|------|----------|----------|
| 行情.csv | 万得代码、时间(HHMMSSmmm)、成交价/量/额、成交笔数、BS标志、当日累计成交量/额、最高/最低/开盘/前收、加权叫买卖价、叫买卖总量、申买/卖价1~10 与量1~10 | 3 秒快照 + 十档盘口，识别资金行为的核心 |
| 逐笔成交.csv | 时间、BS标志('B'买/'S'卖)、成交价格、成交数量 | 逐笔成交，**直接提供主动买卖方向** |
| 逐笔委托.csv | 时间、委托类型('U'撤单/'1'成交/'0'新增)、委托代码('B'/'S')、委托价格、委托数量 | 逐笔委托，**含撤单事件，识别量化的关键** |

### ⚠️ 三个必须记住的数据认知
1. **时间字段是 `HHMMSSmmm` 整数**（如 `93000000` = 9:30:00.000），须转成日内秒数做时段判定，
   不能当 UTC 时间戳。
2. **行情.csv 的 成交量/成交额/成交笔数/当日累计成交量 是当日累计值**，需 diff 才得逐笔量；
   但 **逐笔成交.csv 的 成交数量是逐笔值**，直接 `amount = 价 × 量` 分级，无需 diff。
3. **逐笔成交.csv 的 BS 标志直接给主动买卖方向**（无需靠价格变动推断）；
   **逐笔委托.csv 的 委托类型区分撤单/成交/新增**，是撤单行为特征的核心。

## 四、官方 7 大参考特征集

> 均基于当日盘中可获取数据，无未来函数风险。仅供参考，不强制字段对齐。

1. **OSS 大单分级**：超大/大/中/小单的金额、笔数占比，游资基准占比等。
2. **RS 订单时序**：成交间隔变异系数、拆单相似度、订单爆发率、买卖间隔分化等。
3. **CB 撤单系列**：快速撤单占比、买卖撤单分化、撤单间隔变异系数等。
4. **AP 主动成交**：主动买卖占比、连续成交笔数、单边强度、净成交占比等。
5. **OBP 盘口微观**：最优档位挂单、价差穿越、挂单偏移、盘口失衡度等。
6. **PD 价格发现**：多维价格冲击、盘口失衡、成交效率等。
7. **PI 日内时段**：开盘 30 分钟/尾盘 10 分钟成交占比、赫芬达尔集中度、价格波动等。
